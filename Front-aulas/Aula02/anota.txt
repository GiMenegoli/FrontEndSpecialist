Seletor de Class - USAR SEMPRE
criar uma regra, essa regra receberá um nome,
Esse nome será por qualquer elemento da página.
FORMATAÇÃO PADRÃO - REPETIDA
.nome da class {
propriedade: valor;
}

Na tag HTML, usar o atributo global class
<div class="nome da class"></div>

Seletor de ID - EVITAR USAR
você pode usar o atributo id (identificador) para
criar formatações em CSS
#nome do id {
propriedade: valor;
}

Na tag HTML, usar o atributo global id
<div id="nome id"></div>
ID NÃO PODE SER REPETIDO NA PÁGINA HTML
Você não deve ter dois ou mais elementos com o mesmo ID
